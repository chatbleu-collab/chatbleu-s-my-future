# 나의미래는? - 배포 가이드

## ✅ 폴더 구조 (중요!)
이번 버전은 모든 파일이 **한 폴더에 평평하게** 있어요. 하위 폴더 없음.
```
나의미래는/
├── index.html
├── app.css
├── app.js
├── sw.js
├── manifest.json
├── byeolun.svg
├── icon-192.png
├── icon-512.png
└── 배포가이드.md (이 파일)
```

## GitHub Pages 업로드 방법

### 방법 1: 웹 UI에서 통째로 (가장 쉬움)
1. GitHub 저장소 페이지 접속
2. "Add file" → "Upload files"
3. **`나의미래는` 폴더 안의 파일 8개를 모두 선택해서 한 번에 끌어다 놓기**
4. "Commit changes" 클릭

### 방법 2: 폴더째 (한 번에 끝)
1. GitHub 데스크탑 앱 또는 git 명령 사용
2. 폴더 전체를 끌어다 놓기

### 주의
- **파일 8개가 모두 같은 위치(루트)에 있어야 함**
- 이번 버전은 폴더 구조가 없으니 그냥 파일 8개 다 선택해서 올리면 끝
- 한 번에 안 되면 한 파일씩 올려도 OK

## Vercel 추천 (훨씬 쉬움)
1. https://vercel.com 가입
2. "Add New Project" → 폴더 통째로 드래그
3. 1분 후 라이브
4. GitHub처럼 파일별로 올릴 필요 없음

## 업데이트 시 캐시 문제
브라우저가 옛날 화면 보여주면:
- PC: `Ctrl+Shift+R` (Win) / `Cmd+Shift+R` (Mac)
- 모바일: F12 → Application → Service Workers → "Unregister"
- 또는 시크릿 모드로 열기
